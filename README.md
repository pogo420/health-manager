# health-manager
REST API for managing health.

## What it does?
- Provides endpoints to update steps and weight.
- Provides endpoints to get reports generated via ML engines.

## Techstack:

|Component|Tech|
|---|---|
|Webframework | FastApi |
|Ut framework | Pytest |
|DB | Sqlite |
|Recommendation Engines | AI/ML/Neural|
