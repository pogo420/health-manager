# health-manager
Application for managing health.

## What it does?
- Provides endpoints to update steps and weight.
- Provides endpoints to get reports generated via ML engines.

## Techstack:

|Component|Tech|
|---|---|
|Webframework | FastApi |
|DB | Sqlite |
|Recommendation Engines | AI/ML|
